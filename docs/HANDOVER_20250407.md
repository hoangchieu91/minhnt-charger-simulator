# Bàn Giao Firmware STM32G030 — 07/04/2026

## Tóm tắt phiên làm việc
Hoàn tất migration firmware Minhnt Charger từ **STM32F103** sang **STM32G030C8T6** (Cortex-M0+). Firmware đã được biên dịch, nạp và chạy ổn định trên kit thật qua SEGGER J-Link.

## Công việc đã hoàn thành

### 1. Migration STM32F103 → STM32G030
- Cập nhật Makefile: target `cortex-m0plus`, HAL Driver G0xx
- Thay linker script `STM32G030C8Tx_FLASH.ld` (Flash 64KB, RAM 8KB)
- Sửa toàn bộ thanh ghi UART (`ISR/TDR/RDR` thay cho `SR/DR`)
- Sửa ADC (`CHSELR`, quy trình calibrate G0)
- Sửa Flash write (`DOUBLEWORD` thay `HALFWORD`, page-based erase)

### 2. Fix HardFault Startup
- Nguyên nhân: `__libc_init_array` đọc vùng nhớ rác do thiếu section `.init_array/.fini_array`
- Giải pháp: Bypass `__libc_init_array` trong startup assembly (codebase pure C, không cần C++ constructors)

### 3. Tích hợp DLT645 RS485 Asynchronous Polling (COM2)
- Thêm module `meter_polling.c/h` — đọc đồng hồ điện DLT645 qua USART2 (2400-8-E-1)
- Kiến trúc 100% non-blocking (Interrupt-driven TX/RX) — không ảnh hưởng Modbus COM1
- State machine tự xoay vòng: Voltage → Current → Energy → `Meter_Update()`
- GPIO: TX=`PA2`, RX=`PA3`, DE=`PA4`

### 4. Tài liệu
- Tạo `FIRMWARE_REFERENCE_STM32G0.md` + `.html` (multi-tab, Mermaid diagrams)
- Tích hợp đầy đủ: Kiến trúc, Giải thuật, Modbus Register Map, Billing Flow
- Dọn dẹp tài liệu cũ (F103) không còn phù hợp
